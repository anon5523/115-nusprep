# 臺南市 115 年護理人員甄選備考總部

個人化備考工具，整合簡章、懶人包、讀書計畫、考古題庫、時事政策連結。

## 功能

- 📋 **簡章** - 名額、資格、計分、報名應繳資料
- ⚡ **懶人包** - 五大核心主題、必背數字
- 📅 **讀書計畫** - 50天兩輪複習，每日打勾追蹤，從 2026/5/11 開始
- 📝 **考古題庫** - 109、111、112、113、114 年共 250 題，可隨機抽考、收藏錯題
- 🌐 **時事政策** - 衛福部、疾管署、國健署等官方公告連結
- ⏱️ **倒數計時** - 自動計算距離 6/28 考試剩餘天數
- 🖨️ **列印支援** - A4 列印讀書計畫表

## GitHub Pages 發布步驟

1. 建立新的 GitHub Repository（公開或私有皆可）
2. 上傳兩個檔案到 repo 根目錄：
   - `index.html`
   - `questions_data.js`
3. 進入 repo 設定：**Settings** → **Pages**
4. **Source** 選擇 `Deploy from a branch`
5. **Branch** 選擇 `main` (或 `master`)，資料夾選 `/ (root)`
6. 點擊 **Save**
7. 約 1-2 分鐘後，GitHub 會給你網址：`https://你的帳號.github.io/repo名稱/`

## 本地測試

直接用瀏覽器打開 `index.html` 即可使用，所有資料儲存在瀏覽器 localStorage 中。

## 資料來源

- 簡章與考古題：[臺南市政府衛生局護理人員甄試專區](https://health.tainan.gov.tw/list.asp?orcaid=49027783-987D-4D70-A08E-2FB99422F8E0)
- 105、106、107、108、110 年考古題請自行下載並補入 `questions_data.js`

## 補充考古題的方法

打開 `questions_data.js`，依照已有的格式新增題目：

```javascript
{
  y:108,           // 年份
  n:1,             // 題號
  q:"題目內容",
  a:"A",           // 正解（A/B/C/D）
  o:["選項A","選項B","選項C","選項D"],
  t:"預防接種"     // 主題分類
}
```

## 注意事項

- 此工具為個人學習輔助，**正式報考資訊請以衛生局公告為準**
- 「時事政策」分頁為靜態連結清單，因 GitHub Pages 為靜態網站，無法即時抓取網頁
- 學習進度儲存在瀏覽器 localStorage，清除瀏覽器資料會失去紀錄
- 建議定期匯出進度（手動截圖）作為備份
