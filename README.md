## 🚀 GitHub Pages 部署步驟（5 分鐘完成）

### 1. 建立 Repository

1. 登入 [github.com](https://github.com) → 右上角 **＋ → New repository**
2. 取名，例如 `115-nusprep`（名稱會出現在網址中）
3. 選 **Public** 或 Private 皆可 → **Create repository**

### 2. 上傳三個檔案

進入 repo 頁面 → **Add file → Upload files**，將以下三個檔案拖入：

```
index.html          ← 主網站
questions_data.js   ← 考古題庫（需與 index.html 放在同一目錄）
README.md           ← 這份說明（GitHub repo 首頁自動顯示）
```

點擊 **Commit changes**。

### 3. 開啟 GitHub Pages

1. repo 頁面上方 → **Settings**
2. 左側 **Pages**
3. Source 選 `Deploy from a branch`
4. Branch 選 `main`，資料夾選 `/ (root)` → **Save**

### 4. 取得網址

約 1–2 分鐘後頁面顯示網址：

```
https://你的帳號.github.io/115-nusprep/
```

加入手機書籤，隨時可做題。

> ⚠️ **`questions_data.js` 必須與 `index.html` 在同一個資料夾**，否則題庫無法載入。
