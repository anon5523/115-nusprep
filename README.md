## 🚀 部署到 GitHub Pages（5分鐘完成）

### 步驟一：建立 Repository

1. 登入 [github.com](https://github.com)
2. 右上角 **➕ → New repository**
3. Repository name 取個名字，例如 `nursing-exam-2026`
4. 選 **Public** 或 Private 皆可
5. 點擊 **Create repository**

### 步驟二：上傳檔案

把以下兩個檔案上傳到 repo 根目錄：

```
index.html          ← 主要網站檔案
questions_data.js   ← 考古題題庫
```

上傳方式：在 repo 頁面點 **Add file → Upload files**，把兩個檔案拖進去，然後按 **Commit changes**。

### 步驟三：開啟 GitHub Pages

1. 進入 repo 頁面 → 上方 **Settings**
2. 左側選單找 **Pages**
3. **Source** 選 `Deploy from a branch`
4. **Branch** 選 `main`，資料夾選 `/ (root)`
5. 點擊 **Save**

### 步驟四：取得網址

約 1–2 分鐘後，頁面會顯示你的網址：

```
https://你的帳號.github.io/nursing-exam-2026/
```

> 💡 把這個網址加入手機書籤，隨時可以做題目！
